MyFlix App: Case Study

Case Study Overview

MyFlix is a info-movie app, created for people who loves movies and want to know more about cinema like the All Movie Guide. As a user-friendly app, MyFlix is build to provide users with the best possible experience where they can access information about different movies, actors, actresses, directors, genres and add movie(s)to their list of favourites movies.

The project was one of many achievements offered by Career Foundry. The project goal was to teach and prepear students/developers like me to gain a complete knowldge, understanding and experience in Front-end and Back-end development as a Full-Stack Developer, using some of the most common mordern tech stacks, MERN (Express, React, JavaScript, CSS, HTML and Node.js).


Server-Side Approach

For the Back-end of the app, in order to store movies data and users information, a server-side with RESTful API (interface to exchange information securely over the internet) was built with Node.js and Express, which interacts with a database in MongoDB. To access the API the CRUD (create, read, update and delete), including the post method was implemented witht the result given in a JSON format (i.e., a text-based format for Javascript object syntax).The Postman app was also utilized as a recommanded tool to test each executed request such as the (create, read, update, delete, and post) methods

An important step was to include user authentication and authorization logic in the form of basic HTTP and JWT (JSON Web Token), a standard of internet tool of which upon user successful registration, based on their given credentials, generates an encrypted token to prevent any security issues that may occur.


Client-Side Approach

For better/best user experience, the app web page was build with React (a modern Javascript library) and React-Redux (a state manager to guarantee consistent behaviour across server and client logic). Tech stack like Bootstrap, a (Javascript styling library) with pre-definded built-in components was also used for best pratice and responsive design pueposes for the app.

Upon registration or login in the welcome page, users can navigate between the main view of the app, where a movie list and a search tool is provided to quickly find specific movies; a movie details view that shows more information about the movie like Director info, Movies Genre, and Summary each with a separate view providing users with important information about each movies. In the app user can also edit their profile by either deleting their accoun, update current profile data like chaning the password, email address date of birth, or their username. Users can also add and remove movie(s) from the list of favourite movies in the list of movies.


Challenge Faced

Working with react is my goal, so I was excited to build this app from the beginning. The struggles were most related to the backend since it has a new way to work and debug, but I coop with that with the same natural procedure steps: google search for documentation for new frameworks and libraries, google search for specific issues, console logging when it was possible, including debugging function, and asking for help when none of the firsts could solve my problem.

The next steps for this app will be a better styling tool or intensive Bootstrap learning, to improve visual aspects and a more friendly user experience, although all the chosen tools were a great selection from the course.

Another important lesson is to save more time to reach the expectations of the schedule.
